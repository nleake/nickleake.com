Treasury Design System

2021-current

I joined the Treasury Department at the height of the pandemic. The White House was rolling out the American Rescue Plan and the pressure was on to move, fast. The design system I helped create simplifies bureaucracy by providing official branding. It also has 508 compliant components and a practical user guide for different teams. See it in action on treasury.gov or bep.gov. Download it via GitHub or read the user guide.
